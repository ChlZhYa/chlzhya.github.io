# CLAUDE.md

本文件是 Claude Code 的专属指令。项目同时维护 `AGENTS.md` 作为跨工具通用说明，开始任何工作前请先完整阅读 `@AGENTS.md`：其中的产品方向、用户偏好、技术栈、文件结构与实现准则均适用。

下方为补充规则；若与 `AGENTS.md` 冲突，以本文件为准。

## 语言

- UI 文案整体保持中文优先（沿用 `AGENTS.md`）。
- 技术专有名词保留英文原词，不强行中译：如 React、Vite、TypeScript、canvas、Hero、MVP、slug 等。
- Hero 背景 canvas 动画中出现的词、标签、流程名一律用英文呈现（如 input / retrieval / reasoning / generation / synthesis），不译成中文。

## 隐私

- 不得在网站中暴露用户真实个人信息（姓名、网名、可识别 ID、联系方式等）。
- 站点署名、品牌词不得复用用户的个人标识；如确需名称，使用中性虚构占位（如 "Field Notes"、"Wayfarer" 一类非身份性词），并在交付时提示用户确认或替换。

## 标题文案风格

- Hero 主标题等显眼文案，优先用有诗意、有留白的短句暗示主题，而非直接描述功能或罗列模块。
  - 反例：直接罗列模块名（如"学习、远行与自我系统"）。
  - 正例方向：意象式短句（如"行远自迩"、"在路上写下的字"等）。
  - 具体选词由你给出 2–3 个候选，用户最终拍板。
